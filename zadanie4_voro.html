Pytania teoretyczne

  • 1) Kiedy powstał JavaScript (podaj datę).
    JavaScript został stworzony w grudniu 1995 roku.
  • 2) Dlaczego jest to język skryptowy?
    Jest to język skryptowy, ponieważ kod jest interpretowany na bieżąco przez przeglądarkę, a nie kompilowany do kodu maszynowego przed uruchomieniem.
  • 3) Gdzie jest wykonywany JS (klient lub serwer)?
    Głównie jest wykonywany po stronie klienta, czyli w przeglądarce internetowej.
  • 4) Dlaczego jest JS bezpiecznym narzędziem?
    JS jest bezpieczny, ponieważ działa w izolowanym środowisku zwanym "piaskownicą" (sandbox), co uniemożliwia mu bezpośredni dostęp do systemu plików użytkownika.
  • 5) Czy jest to język obiektowy, jeśli tak to dlaczego.
    Tak, jest to język obiektowy, ponieważ wszystko w nim, poza typami prostymi, jest traktowane jako obiekt.
  • 6) Wymień zalety JS:
    • **Zaleta 1:** Zwiększa interaktywność stron internetowych.
    • **Zaleta 2:** Jest powszechnie wspierany przez wszystkie nowoczesne przeglądarki, co zapewnia jego uniwersalność.
    • **Zaleta 3:** Umożliwia tworzenie dynamicznych treści bez konieczności przeładowywania strony.
  • 7) Framework to:
    Framework to ustrukturyzowany zbiór narzędzi, bibliotek i reguł, który dostarcza gotowy "szkielet" do tworzenia aplikacji, ułatwiając i przyspieszając pracę.
  • 8) Do aplikacji mobilnych można użyć Framework
    Tak, można używać frameworków, np. React Native, Ionic czy Flutter, do tworzenia aplikacji mobilnych na różne platformy.
  • 9) Aplikacje responsywne to:
    Aplikacje responsywne to te, które automatycznie dostosowują swój wygląd i układ do rozmiaru ekranu urządzenia, na którym są wyświetlane.
  • 10) Przepisz linie kodu pod nimi wytłumaczenie jak działa ten fragment kodu
    `<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen('')">`
    Jest to przycisk, który po kliknięciu wywołuje funkcję JavaScript o nazwie `WinOpen('')`, która otwiera nową stronę.
  • 11) Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
    `window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");`
    Ta instrukcja otwiera nowe okno.
    • `toolbar=no` - ukrywa pasek narzędzi.
    • `directories=no` - ukrywa przyciski katalogów.
    • `menubar=no` - ukrywa pasek menu.
    • `height=280` - ustawia wysokość okna na 280 pikseli.
    • `width=160` - ustawia szerokość okna na 160 pikseli.
    • `top=200` - ustala odległość okna od górnej krawędzi ekranu.
    • `left=200` - ustala odległość okna od lewej krawędzi ekranu.
  • 12) Opisz następującą instrukcję window.close()
    Instrukcja ta służy do programowego zamknięcia bieżącego okna przeglądarki.
