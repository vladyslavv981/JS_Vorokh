Pytania

1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

3. Wymień cechy charakterystyczne konstruktora.

4. Konstruowanie obiektu – obiekt ma trzy pola oraz jedną metodę.

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

7. Znaczenie słowa kluczowego new wraz z przykładem.

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Odpowiedzi

1. Określanie właściwości obiektu

Sposób 1 – notacja kropkowa:


auto.marka = "Ferrari";
auto.rok = 2003;
auto.cena = 200000;

Sposób 2 – notacja tablicowa:


auto["cena"] = 200000;

2. Odwoływanie się do pól i metod

Do właściwości i metod odwołujemy się przez nazwę obiektu oraz kropkę. 


console.log(auto.marka);
auto.jedz();

3. Cechy charakterystyczne konstruktora

  • Jest wywoływany automatycznie podczas tworzenia obiektu
  • Nazywa się tak samo jak klasa
  • Nie zwraca żadnej wartości
  • Może przyjmować argumenty
  • Do jego wywołania używa się słowa kluczowego new

4. Konstruowanie obiektu – przykład


var osoba = {
    nazwisko: 'Vorokh',
    imie: 'Vladyslav',
    zawod: 'programista',
    wyswietl: function() {
        document.write(this.imie + ' ' + this.nazwisko);
    }
}

5. Znaczenie słowa this

Słowo kluczowe this pozwala odwołać się do pól i metod obiektu z jego wnętrza.

Pola i metody w obiekcie oddzielamy przecinkami.

6. Konstruktor w JavaScript


function Osoba(imie, nazwisko, zawod) {
    this.imie = imie;
    this.nazwisko = nazwisko;
    this.zawod = zawod;
    this.wypisz = function() {
        alert(this.imie + ' ' + this.nazwisko);
    }
}

Funkcja pełni rolę klasy, a słowo this wskazuje na aktualnie tworzony obiekt.

7. Słowo kluczowe new

Słowo new służy do tworzenia nowego obiektu na podstawie konstruktora. 


var osoba1 = new Osoba('Vladyslav', 'Vorokh', 'uczen');
var osoba2 = new Osoba('Anna', 'Nowak', 'sekretarka');

8. Właściwość prototype

Prototype umożliwia dodawanie wspólnych metod i właściwości dla wszystkich obiektów utworzonych na podstawie danego konstruktora. 


function Klient() {
    this.imie = 'Vladyslav';
    this.nazwisko = 'Vorokh';
}

Klient.prototype.zawod = 'programista';

Klient.prototype.pisz_dane = function() {
    document.write(this.imie + ' ' + this.nazwisko + ' - ' + this.zawod);
}

var osoba = new Klient();
osoba.pisz_dane();

Po użyciu prototype każdy nowy obiekt utworzony z konstruktora Klient dziedziczy dodane metody i właściwości.
